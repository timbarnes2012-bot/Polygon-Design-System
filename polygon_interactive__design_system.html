Polygon Interactive Offline Design SystemStandalone and editable
Personal digital design language

Refined interfaces with warmth, clarity and quiet authority.

A reusable system for websites, landing pages and software. It combines fabric-inspired ivory surfaces, a distinct Roman Clay material texture, light grey, layered dark neutrals, softened blues and precise brass detailing.

Foundations

Colour system

Ivory forms the dominant field. Light grey remains a clean neutral colour, while Roman Clay is a separate tactile material for selected banners and backgrounds. Navy and charcoal provide structure, the lighter pale blue supports calm surfaces, French blue signals interaction and brass is reserved for restrained detailing.

Recommended balance

57% clean light neutrals, about 8% Roman Clay material, 21% structural darks, 12% blues and 2–4% brass.

AccessibilityUse navy or charcoal text on ivory. Reserve brass for borders, icons and highlights rather than small body text.
Foundations

Typography

Fraunces is specified for titles and headings. Inter is specified for UI and body copy. JetBrains Mono is specified for labels, code and data. The file remains offline and uses locally installed versions of those fonts, with reliable system fallbacks when they are unavailable.

Designed with restraint.

Titles and headings: Fraunces · UI and body: Inter · Labels, code and data: JetBrains Mono

Use generous spacing, short paragraphs and decisive hierarchy. Interfaces should feel composed, tactile and sophisticated rather than crowded or decorative.

Foundations

Spacing, shape and elevation

Spacing

4, 8, 12, 16, 24, 32, 48, 64 and 96px.

Radii

8px controls, 15px cards and 26px feature panels.

Elevation

Use soft navy-tinted shadows. Avoid sharp black drop shadows.

Roman clay surface

Roman Clay backgrounds should feel mineral, soft and tactile.

Use the supplied Roman Clay material for banners, feature strips, quiet section backgrounds and selected presentation panels. Avoid applying it behind dense tables or long-form body copy.

Heading tabs

Components

Buttons

Use the lighter French Blue for standard actions, with dark brass-brown text for contrast. Use Pale Blue for lower-emphasis actions and heading tabs, and Navy Blue only when the user is confirming, saving, submitting or completing an important workflow.

Components

Forms

Form guidance

Keep labels visible, use a minimum 44px control height, show focus clearly and never rely on colour alone to communicate an error.

Helpful messageUse plain language and place validation beside the affected field.
